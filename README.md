# JotForm API Client Library for PHP

## Description

A simple [JotForm API](http://api.jotform.com/docs/) library for PHP.

## Requirements

- [PHP >= 5.4](http://php.net/)
- [Guzzle 6](https://github.com/guzzle/guzzle)
